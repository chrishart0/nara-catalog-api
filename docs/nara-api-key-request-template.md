# NARA Catalog API Key Request Template

Send to: `Catalog_API@nara.gov`

Subject: NARA Catalog API key request

```text
Hello,

I would like to request a National Archives Catalog API key.

Intended use:
I plan to use the API for read-only research workflows, including searching
Catalog records, retrieving record metadata by NAID, listing available digital
objects, and preserving source metadata for citation/audit purposes.

Expected volume:
I expect light to moderate use within the standard monthly limit.

Project/tool:
I am using the NARA Catalog Agent CLI:
https://github.com/chrishart0/nara-catalog-api

Name:
[Your name]

Organization, if applicable:
[Your organization]

Email:
[Your email]

Thank you.
```

NARA commonly issues keys with a default limit of 10,000 API calls per month.
In project use, responses have often arrived within about one business day, but
response time is not guaranteed.
